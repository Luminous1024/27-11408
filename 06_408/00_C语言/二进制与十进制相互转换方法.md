# 二进制与十进制相互转换方法

## 一、**二进制转十进制**

### 方法：**加权求和法**
二进制数的每一位乘以对应的 2 的幂次（权重），然后求和。

**公式：**
$$
\text{十进制数} = \sum_{i=0}^{n} b_i \times 2^i
$$
其中 \( b_i \) 是二进制数的第 i 位（从右向左，从 0 开始编号）

### **步骤：**
1. 从二进制数**最右边**开始（最低位），给每位编号，从 0 开始
2. 每位数字乘以 \( 2^{\text{位索引}} \)
3. 将所有乘积相加

### **示例：**
将二进制数 `1011` 转换为十进制：

```
二进制：  1    0    1    1
位索引：  3    2    1    0   ← 从右向左编号
权重：   2³   2²   2¹   2⁰

计算：1×2³ + 0×2² + 1×2¹ + 1×2⁰
    = 1×8  + 0×4  + 1×2  + 1×1
    = 8    + 0    + 2    + 1
    = 11
```
∴ `1011`₂ = `11`₁₀

### **快速记忆表：**
| 二进制位权重 | 2⁷  | 2⁶  | 2⁵  | 2⁴  | 2³  | 2²  | 2¹  | 2⁰  |
| ------ | --- | --- | --- | --- | --- | --- | --- | --- |
| 十进制值   | 128 | 64  | 32  | 16  | 8   | 4   | 2   | 1   |

**练习：** `110101`₂ = ?
```
1×2⁵ + 1×2⁴ + 0×2³ + 1×2² + 0×2¹ + 1×2⁰
= 32 + 16 + 0 + 4 + 0 + 1
= 53
```

## 二、**十进制转二进制**

### **方法1：除2取余法（整数部分）**
适用于**十进制整数**转二进制整数。

**步骤：**
1. 将十进制整数除以 2，记录商和余数
2. 用商继续除以 2，再记录余数
3. 重复直到商为 0
4. **将余数从下往上（逆序）排列**

### **示例：** 将 `29` 转换为二进制
```
29 ÷ 2 = 14 ... 余 1  ↑
14 ÷ 2 = 7  ... 余 0  | 逆序
7  ÷ 2 = 3  ... 余 1  | 排列
3  ÷ 2 = 1  ... 余 1  |
1  ÷ 2 = 0  ... 余 1  ↓
```
从下往上读取余数：`11101`
∴ `29`₁₀ = `11101`₂

### **方法2：乘2取整法（小数部分）**
适用于**十进制纯小数**转二进制小数。

**步骤：**
1. 将小数部分乘以 2
2. 记录整数部分（0 或 1）
3. 取结果的小数部分继续乘以 2
4. 重复直到小数部分为 0 或达到所需精度
5. **将整数部分从上往下（正序）排列**

### **示例：** 将 `0.625` 转换为二进制
```
0.625 × 2 = 1.25  → 整数部分 1，取小数 0.25  ↓
0.25  × 2 = 0.5   → 整数部分 0，取小数 0.5   正序
0.5   × 2 = 1.0   → 整数部分 1，取小数 0.0   排列
```
从上往下读取整数部分：`101`
∴ `0.625`₁₀ = `0.101`₂

### **方法3：拆分法（利用2的幂次）**
适用于熟悉 2 的幂次的快速转换。

**示例：** 将 `45` 转换为二进制
```
45 = 32 + 8 + 4 + 1
   = 2⁵ + 2³ + 2² + 2⁰
对应二进制位： 1  0  1  1  0  1
位索引：       5  4  3  2  1  0
```
从高位到低位：`101101`₂

## 三、**带小数的十进制数转换**
整数部分和小数部分**分开转换**，然后合并。

**示例：** 将 `25.375` 转换为二进制
1. 整数部分 `25`：
   ```
   25 ÷ 2 = 12 ... 1
   12 ÷ 2 = 6  ... 0
   6  ÷ 2 = 3  ... 0
   3  ÷ 2 = 1  ... 1
   1  ÷ 2 = 0  ... 1
   ```
   逆序余数：`11001`
2. 小数部分 `0.375`：
   ```
   0.375 × 2 = 0.75 → 0
   0.75  × 2 = 1.5  → 1
   0.5   × 2 = 1.0  → 1
   ```
   正序整数：`011`
3. 合并：`11001.011`₂

## 四、**特殊情况和注意事项**

### 1. **二进制补码表示负数**
计算机中常用补码表示负数：
- 正数：直接转换
- 负数：先表示绝对值的二进制，然后取反加 1

### 2. **精度问题**
某些十进制小数无法精确转换为二进制有限小数（如 0.1₁₀）：
```
0.1 × 2 = 0.2 → 0
0.2 × 2 = 0.4 → 0
0.4 × 2 = 0.8 → 0
0.8 × 2 = 1.6 → 1
0.6 × 2 = 1.2 → 1
...
0.1₁₀ = 0.0001100110011...₂（无限循环）
```

### 3. **快速检查**
- 二进制数的**最后一位**：为 0 表示偶数，为 1 表示奇数
- **二进制转十进制**时，每个 1 的位置代表一个 2 的幂次
- **十进制转二进制**时，不断除以 2 相当于不断“提取”二进制位

## **总结表格**

| 转换方向            | 方法     | 关键步骤        | 排列顺序     |
| --------------- | ------ | ----------- | -------- |
| **二进制 → 十进制**   | 加权求和   | 每位 × 2ⁿ 后相加 | —        |
| **十进制整数 → 二进制** | 除 2 取余 | 连续除以 2 记录余数 | 余数**逆序** |
| **十进制小数 → 二进制** | 乘 2 取整 | 连续乘以 2 记录整数 | 整数**正序** |

**记忆口诀：**
- 二转十：**按权展开，求和得数**
- 十转二（整数）：**除二取余，逆序排列**
- 十转二（小数）：**乘二取整，正序排列**
