# C语言32个关键字详解表

| 序号 | 关键字 | 类别 | 含义与用途 | 示例 |
|------|--------|------|-----------|------|
| 1 | `auto` | 存储类别 | 声明自动变量（局部变量默认） | `auto int x = 10;` |
| 2 | `break` | 流程控制 | 跳出循环或switch语句 | `while(1) { break; }` |
| 3 | `case` | 流程控制 | switch语句的分支标签 | `case 1: printf("one");` |
| 4 | `char` | 数据类型 | 字符类型（1字节） | `char c = 'A';` |
| 5 | `const` | 类型修饰 | 定义常量，值不可修改 | `const int MAX = 100;` |
| 6 | `continue` | 流程控制 | 跳过当前循环的剩余部分 | `for(i=0;i<10;i++){if(i==5)continue;}` |
| 7 | `default` | 流程控制 | switch语句的默认分支 | `default: printf("other");` |
| 8 | `do` | 流程控制 | do-while循环的开始 | `do { x++; } while(x<10);` |
| 9 | `double` | 数据类型 | 双精度浮点类型（8字节） | `double pi = 3.14159;` |
| 10 | `else` | 流程控制 | if语句的否定分支 | `if(x>0) {...} else {...}` |
| 11 | `enum` | 数据类型 | 枚举类型定义 | `enum Color {RED, GREEN, BLUE};` |
| 12 | `extern` | 存储类别 | 声明外部全局变量/函数 | `extern int global_var;` |
| 13 | `float` | 数据类型 | 单精度浮点类型（4字节） | `float f = 3.14f;` |
| 14 | `for` | 流程控制 | for循环语句 | `for(i=0; i<10; i++) {...}` |
| 15 | `goto` | 流程控制 | 无条件跳转语句 | `goto label; ... label: ...` |
| 16 | `if` | 流程控制 | 条件判断语句 | `if(x > 0) { ... }` |
| 17 | `int` | 数据类型 | 整型（通常4字节） | `int count = 10;` |
| 18 | `long` | 数据类型 | 长整型 | `long big = 100000L;` |
| 19 | `register` | 存储类别 | 建议编译器将变量放入寄存器 | `register int i;` |
| 20 | `return` | 流程控制 | 函数返回值 | `return x + y;` |
| 21 | `short` | 数据类型 | 短整型（通常2字节） | `short s = 100;` |
| 22 | `signed` | 类型修饰 | 有符号类型（默认） | `signed int x = -10;` |
| 23 | `sizeof` | 运算符 | 获取数据类型或对象的大小 | `sizeof(int); sizeof(arr);` |
| 24 | `static` | 存储类别 | 静态变量/函数 | `static int counter = 0;` |
| 25 | `struct` | 数据类型 | 结构体类型定义 | `struct Point { int x; int y; };` |
| 26 | `switch` | 流程控制 | 多路分支选择语句 | `switch(grade) { case 'A': ... }` |
| 27 | `typedef` | 类型定义 | 为现有类型定义新名称 | `typedef int Integer;` |
| 28 | `union` | 数据类型 | 联合体类型定义 | `union Data { int i; float f; };` |
| 29 | `unsigned` | 类型修饰 | 无符号类型 | `unsigned int u = 100;` |
| 30 | `void` | 数据类型 | 无类型/空类型 | `void func(void);` |
| 31 | `volatile` | 类型修饰 | 易变变量（防止优化） | `volatile int flag;` |
| 32 | `while` | 流程控制 | while循环语句 | `while(x > 0) { x--; }` |

## **使用频率统计**

| 频率等级 | 关键字 | 说明 |
|----------|--------|------|
| **高频** | `int`, `if`, `for`, `while`, `return`, `char`, `void` | 几乎每个程序都会使用 |
| **中频** | `struct`, `float`, `double`, `switch`, `case`, `break`, `continue` | 常用但非必需 |
| **低频** | `auto`, `register`, `goto`, `union`, `volatile` | 特定场景使用 |
| **极少用** | `_Bool`, `_Complex`, `_Imaginary` | 特殊需求使用 |

## **注意事项**

1. **关键字不能用作标识符**（变量名、函数名等）
2. **C语言区分大小写**：`int`是关键字，`INT`不是
3. **不同标准的关键字可能不同**：C99/C11增加了新关键字
4. **某些编译器可能支持扩展关键字**：如`__asm__`, `__attribute__`等
5. **C++的关键字更多**：C语言程序员学习C++时需注意差异

## **示例代码：展示常用关键字**

```c
#include <stdio.h>

// 数据类型和存储类别
int global_var;                    // int类型，全局变量
static int static_var = 10;        // static存储类别
const float PI = 3.14159f;         // const常量

// 结构体和枚举
struct Point {                     // struct关键字
    int x;
    int y;
};

enum Color {                       // enum关键字
    RED,
    GREEN,
    BLUE
};

// 函数声明和定义
void print_message(void);          // void返回类型，void参数
static void internal_func(void);   // static函数

int main(void) {                   // 主函数
    // 变量声明
    auto int local_var = 5;        // auto关键字（可省略）
    register int counter;          // register关键字
    volatile int flag = 0;         // volatile关键字
    
    // 流程控制
    if (local_var > 0) {           // if-else
        printf("Positive\n");
    } else {
        printf("Non-positive\n");
    }
    
    // switch-case
    switch (local_var) {           // switch关键字
        case 1:                    // case关键字
            printf("One\n");
            break;                 // break关键字
        case 5:
            printf("Five\n");
            break;
        default:                   // default关键字
            printf("Other\n");
    }
    
    // 循环
    for (counter = 0; counter < 5; counter++) {  // for循环
        if (counter == 2) {
            continue;              // continue关键字
        }
        printf("Counter: %d\n", counter);
    }
    
    while (flag == 0) {            // while循环
        flag = 1;
    }
    
    do {                           // do-while循环
        printf("Do-while\n");
    } while (0);
    
    // goto（谨慎使用）
    goto end;                      // goto关键字
    
end:
    printf("Program end\n");
    return 0;                      // return关键字
}

// sizeof运算符示例
void size_demo(void) {
    printf("Size of int: %zu\n", sizeof(int));      // sizeof关键字
    printf("Size of double: %zu\n", sizeof(double));
}

// typedef示例
typedef unsigned int UINT;         // typedef关键字
typedef struct Point POINT;

// union示例
union Data {                        // union关键字
    int i;
    float f;
    char str[20];
};
```
