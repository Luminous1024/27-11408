## 💡 使用提示

1. **优先级记忆口诀**：
   - 括号成员第一，全体单目第二
   - 乘除余三，加减四
   - 移位五，关系六
   - 等与不等排第七
   - 位与异或和位或，八九十
   - 逻辑与或十一二
   - 条件高于赋值，逗号运算最低级

2. **重要说明**：
   - 优先级相同则看结合性（从左到右或从右到左）
   - 不确定时**使用括号**明确优先级
   - 后缀++/--优先级高于前缀++/--
   - 位运算符优先级：& > ^ > |

3. **常用记忆技巧**：
   ```c
   // 易混淆的例子
   *p++      // 等价于 *(p++)，先取指针值，再指针++
   (*p)++    // 先解引用，再值++
   a & b == c // 等价于 a & (b == c)，注意位与优先级低于比较！
   ```
